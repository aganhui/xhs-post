# 发布故障排查

## 常见问题

### 1. 发布返回 402

**原因**：余额不足，发布需要 ¥2.00。

**处理**：告知用户余额不足，建议充值或联系 admin。

### 2. 上传图片失败

**可能原因**：
- 图片文件不存在（检查 /tmp/xhs-cards/ 路径）
- 图片超过 10MB 限制
- note_id 不正确
- 网络问题

**处理**：检查文件路径和大小，重试上传。

### 3. 二维码无法扫码

**可能原因**：
- myaibot 服务暂时不可用
- 小红书 App 版本过低

**处理**：尝试用 publish_url 直接在小红书 App 中打开。

### 4. 标题被截断

**原因**：myaibot 标题权重限制为 20（中文=1，英文/数字=0.5）。

**处理**：在创建笔记前用 xhs-copy 的标题权重规则检查，超重则缩短标题。

### 5. 图片显示不全

**原因**：上传的图片 URL 失效或格式不支持。

**处理**：确保图片是 PNG/JPG 格式，重新上传。

## API Key 问题

### Key 无效 (401)

检查步骤：
1. `echo $XHS_POST_API_KEY` — 确认环境变量已设置
2. 确认 Key 没有多余空格或引号
3. 用 `/api/me` 验证 Key 是否有效

### Key 未设置

```bash
# 写入 ~/.zshrc 持久化
echo 'export XHS_POST_API_KEY="your-key"' >> ~/.zshrc
source ~/.zshrc
```
